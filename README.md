# spotify_house_party
An app where you can invite your friends and jam together.

# Installing Dependencies.

- Python and Node.js should be installed.

## Django dependencies.
``` pip install -r requirements.txt ```

## JS dependencies.
```
cd frontend/
npm i
```

# Starting the App

## Building React pages.
```
cd frontend/
npm run dev
```

## Starting django
```
python manage.py runserver
```
